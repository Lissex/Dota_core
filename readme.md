# Dota_core

Это мой первый проект на Python, сделанный примерно год назад.  
Проект представляет собой Телеграм-бота на библиотеке `aiogram` с подключением к базе данных.

## Структура проекта

```
Dota_core/
├── bot/                # Основной код бота
├── database/           # Файлы и скрипты базы данных
├── .venv/              # Виртуальное окружение Python
└── README.md           # Этот файл
```

## Установка

1. Клонируйте репозиторий:
```bash
git clone <URL_репозитория>
cd Dota_core
```

2. Создайте виртуальное окружение (если его нет):
```bash
python -m venv .venv
```

3. Активируйте виртуальное окружение:

- Windows:
```bash
.venv\Scripts\activate
```

- Linux/MacOS:
```bash
source .venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` и добавьте токен бота:

```env
BOT_TOKEN=ваш_токен
```

При необходимости настройте подключение к базе данных в соответствующих файлах конфигурации.

## Запуск

```bash
python bot/main.py
```

## Используемые технологии

- Python 3.13+
- aiogram
- SQLAlchemy / aiosqlite (для работы с базой данных)
- aiohttp, requests и другие зависимости

## Особенности проекта

- Асинхронная обработка сообщений с помощью `aiogram`
- Работа с базой данных для хранения информации
- Легкая структура для расширения функционала бота
